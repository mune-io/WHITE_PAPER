1. Taxi Advertising System

Status: Completed ✅

A digital advertising network with tablets installed in taxis. Each screen can also serve as a computing node for distributed processing in the Web3Taxi system.

2. Decentralized Taxi App (Web3Taxi)

Status: In Progress ⚙️

A peer-to-peer system where ride processing occurs directly on devices — tablets, mobile phones, or mini-servers.

  • libp2p + WebRTC — peer-to-peer communication
  • Redis / Gossip — state synchronization
  • DID — decentralized identification
  • Python / Go / React Native — implementation across devices

3. Offline Network (Meshtastic / LoRa)

Status: Planned 🔄

Using Meshtastic mesh networking enables offline communication without internet or telecom operators, covering entire cities through LoRa connectivity.

4. Internal Payment System

Status: Planned 🔄

Introducing a tokenized economy for transactions between drivers, passengers, and advertisers, built on Polygon or Base blockchain.

Conclusion & Disclaimer

This ecosystem merges transportation, advertising, and computation into one decentralized network. All calculations are indicative and not investment advice.
